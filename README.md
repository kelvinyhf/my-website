# My Website
A clean, modern, and responsive website built from scratch to showcase my coding journey and skills.

![Stars](https://img.shields.io/github/stars/kelvinyhf/my-website)
![Forks](https://img.shields.io/github/forks/kelvinyhf/my-website)

> “Your stars keep me motivated!"

## What’s Inside?
- **About Me**: A brief introduction to who I am.
- **Timeline**: My coding journey from writing my first line of code to diving into web development.
- **Experience**: A showcase of my frontend tech stack with visual proficiency bars.
- **AI Chat**: An interactive chatbot powered by structured system prompts, acting as a digital version of myself.

## Features
- **Fully Responsive**: Seamless experience across mobile, tablet, and desktop screens.
- **Theme Detection**: Supports both light and dark modes.
