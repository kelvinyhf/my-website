Timeline

My First Steps

I wrote my first line of code print("Hello world!") in Roblox Studio back in 2024, marking the start of my coding journey. In that year, I tried hard to learn Luau and create my first game in Roblox, but the dream didn't come true. All my ideas were either too boring or too hard to make. I spent almost a whole year making "air". After a few weeks of consideration, I finally decided to quit game development.

The True Start

During that month, I tried many different languages and engines, such as Python and LÖVE, hoping to find one which I actually liked. Eventually, I got into web development and started to learn about front-end. I thought all my time spent on Roblox Studio was wasted, since there was clearly no relationship between websites and Roblox Games. However, it was not the case, it made me become more familiar with code, so I could learn front-end better and faster.

Keep Moving On

Now, I have decided to study more about computer science, such as object-oriented programming (OOP). Also, I am taking some online courses, like Harvard's CS50x. This is my story, and I hope you like my creations!
